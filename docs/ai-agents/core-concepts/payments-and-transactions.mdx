Your agent has a wallet and a verified identity. Now it needs to act.
Onchain actions are driven by skills. Skills give your agent tools to interact with X402 payments, smart contracts, and other services.

## How agents transact

Agents build their own transactions based on the prompts defined in skill files. The skill file tells the agent what actions are available, what parameters are required, and how to construct the transaction. The agent interprets these prompts and executes accordingly.

## X402

X402 is an open payment protocol for instant stablecoin payments over HTTP. Your agent requests a resource, pays programmatically, and receives access.
[Deep dive on X402 →](/learn/onchain-concepts/x402)

## Skills

Skills are tools your agent can call. If you've used Claude or other LLMs with tool use, it's the same concept - predefined capabilities the agent can invoke when needed.
Each skill defines what it does, what inputs it expects, and what the agent can do with it. Your agent reads the skill, understands its purpose, and calls it when relevant to the task.
